# IPL-Cricket-Data-Analysis
# 🏏 IPL Match Analysis 

## 📌 Project Overview
This project is based on the Indian Premier League (IPL) data analysis.

## 🎯 Project Goal
- Analyze IPL match data
- Find important insights (teams, players, performance)

## 🧠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## 📊 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Team performance analysis
- Player statistics analysis

## 📁 Dataset
The dataset contains IPL match details such as:
- Teams
- Venue
- Toss winner
- Match result
- Player performance
