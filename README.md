# IPL-Cricket-Data-Analysis
#  IPL Data Analysis (EDA Project)

##  Project Overview
This is a practice project based on the Indian Premier League (IPL) dataset.  
In this project, Exploratory Data Analysis (EDA) is performed to understand patterns, trends, and insights from IPL matches.

##  Project Goal
- Understand IPL dataset using data analysis
- Perform data cleaning and preprocessing
- Extract useful insights from match data
- Visualize data using graphs

##  Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

##  Analysis Performed
- Data Cleaning (handling missing values)
- Understanding dataset structure
- Team performance analysis
- Toss impact analysis
- Match result analysis
- Data visualization using charts and graphs

##  Dataset
- IPL dataset containing match details
- Includes teams, venue, toss, result, etc.
- Dataset size: 17MB (included in repository)

##  Visualizations
- Bar charts
- Pie charts
- Count plots
- Heatmaps (if used)
